Original Article: https://www.bleepingcomputer.com/news/microsoft/microsoft-releases-windows-11-oob-hotpatch-to-fix-rras-rce-flaw/

1) Microsoft has issued an out-of-band hotpatch update, KB5084597, for select Windows 11 Enterprise versions to address remote code execution vulnerabilities discovered in the Windows Routing and Remote Access Service (RRAS). These vulnerabilities, identified as CVE-2026-25172, CVE-2026-25173, and CVE-2026-26111, permit an attacker to execute remote commands by tricking authenticated domain users into sending requests to malicious servers. While a fix was part of March 2026 updates requiring a system reboot, the hotpatch version provides an in-memory fix for systems critical to operations, bypassing the need for immediate reboots.

2) This update matters because it addresses RCE vulnerabilities in systems where reboots are disruptive, ensuring they remain secure against exploitation without downtime. Attackers leveraging these vulnerabilities could potentially gain remote access or control over systems, posing significant security and operational risks. By providing a seamless patching mechanism, businesses with mission-critical services, especially those with complex server management needs, are better protected against immediate threats without compromising uptime.

3) Organizations should review and verify if they are part of the hotpatch update program and ensure their eligible Windows 11 Enterprise systems are managed through Windows Autopatch. This guarantees the prompt application of the hotpatch without manual intervention. Additionally, it is prudent to continue monitoring Microsoft's security advisories and potentially adjust patch management strategies to include hotpatch updates, especially for systems essential to continuous operations. Network administrators should also ensure user training includes recognizing phishing and other deceptive tactics that could exploit such RCE vulnerabilities.
