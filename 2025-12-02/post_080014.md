Original Article: https://www.bleepingcomputer.com/news/security/glassworm-malware-returns-in-third-wave-of-malicious-vs-code-packages/

**1) What happened:**  
The Glassworm malware campaign has launched its third wave, infiltrating the OpenVSX and Microsoft Visual Studio marketplaces with 24 new malicious packages. These packages are disguised as extensions for VS Code editors, with malicious code using "invisible Unicode characters" to conceal itself. After installation, the malware attempts to exfiltrate user credentials and cryptocurrency wallet data and enables remote access through HVNC. Despite past containment efforts, Glassworm reappeared with deceptive publisher accounts and evolved technically, now using Rust-based implants and inflating download counts to enhance credibility.

**2) Why it matters:**  
This resurgence of Glassworm underscores persistent vulnerabilities in software supply chains, particularly within trusted extension repositories. It affects developers globally by compromising the tools they rely on, potentially breaching sensitive data like GitHub and cryptocurrency accounts. This type of attack can lead to wider-scale disruptions, losses, and trust issues within software development communities, necessitating enhanced vigilance. The use of obfuscation techniques and counterfeit popularity highlights the sophistication of modern malware, presenting significant detection and remediation challenges for platform operators.

**3) What actions should be taken:**  
Development and security teams should enhance their vigilance regarding third-party extensions, especially scrutinizing packages for unusual activity or obfuscation techniques like invisible Unicode characters. Organizations could implement stricter controls over the installation of third-party software and use isolated environments to test extensions. Repository operators like Microsoft and OpenVSX are encouraged to bolster their screening processes, perhaps leveraging machine learning to spot anomalous patterns. Users could utilize features like "Report Abuse" links to alert these platforms to suspicious behaviors, thereby contributing to community security efforts.
