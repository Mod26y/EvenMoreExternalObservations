Original Article: https://www.bleepingcomputer.com/news/microsoft/microsoft-adds-windows-protections-for-malicious-remote-desktop-files/

1) **What happened:** Microsoft has rolled out new protections for Windows 10 and 11 to safeguard against misuse of Remote Desktop Protocol (RDP) files that are often targeted in phishing attacks. These files can redirect local resources to remote hosts, a function that has been exploited by hackers, notably the Russian APT29 group, to access and steal data from victims. The update introduces educational prompts, a security dialog for first-time RDP file openings, and default disabling of risky resources to enhance user security.

2) **Why it matters:** The new protections are crucial as RDP files are a favorite tool for cybercriminals, allowing unauthorized access to sensitive systems and data in a stealthy manner. The enhanced security measures address significant vulnerabilities by educating users and reducing the risk of exploitation through unverified connections. This development is particularly important for enterprise environments where remote desktop functionalities are extensively used.

3) **What actions should be taken:** Organizations should ensure these updates are installed promptly to benefit from the enhanced RDP file protections. Additionally, training employees on recognizing phishing threats and handling RDP files securely is advised. IT departments should keep these security features enabled, avoiding temporary disablements unless absolutely necessary. Continuous monitoring for any related threats and anomalies should complement these efforts, ensuring robust defense against potential attacks.
