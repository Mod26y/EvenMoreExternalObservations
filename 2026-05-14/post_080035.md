Original Article: https://www.bleepingcomputer.com/news/security/windows-bitlocker-zero-day-gives-access-to-protected-drives-poc-released/

1) What happened: A researcher known as Chaotic Eclipse publicly disclosed two unpatched vulnerabilities in Microsoft's Windows systems: YellowKey, which allows BitLocker bypass, and GreenPlasma, a privilege-escalation flaw. YellowKey can be executed through the Windows Recovery Environment, granting unauthorized access to BitLocker-protected drives without credentials. On the other hand, GreenPlasma potentially elevates privileges to obtain SYSTEM permissions. Both exploits were revealed due to dissatisfaction with Microsoft's bug reporting and response process, emphasizing vulnerabilities in Windows 11 and Windows Server 2022/2025.

2) Why it matters: These zero-day vulnerabilities create significant security risks for Windows users and organizations relying on BitLocker for drive encryption and data security. If exploited, YellowKey could enable unauthorized access to sensitive data, undermining trust in TPM-backed encryption. GreenPlasma's potential to elevate privileges could lead to unauthorized actions by attackers, compromising system integrity. Public disclosure without immediate patches from Microsoft gives attackers a potential window of opportunity, highlighting the critical impact of effective vulnerability management and prompt vendor response to security reports.

3) What actions should be taken: Organizations should immediately review BitLocker configurations, especially TPM-only setups, and consider adding a BitLocker PIN and BIOS password as temporary mitigations, acknowledging that these may not fully prevent exploitation. System administrators should keep abreast of Microsoft's forthcoming security patches and apply them promptly when available. Additionally, enhanced monitoring for potential exploit activity and educating users on safe USB usage can be effective interim measures. Engaging with vulnerability management programs that emphasize swift patching and testing may prevent future exposure to similar risks.
