Original Article: https://www.bleepingcomputer.com/news/security/critical-everest-forms-pro-flaw-exploited-to-take-over-wordpress-sites/

**1) What happened:**

A critical vulnerability, identified as CVE-2026-3300, has been discovered in the Everest Forms Pro plugin for WordPress, which allows hackers to gain complete control over affected websites. The flaw affects versions 1.9.12 and earlier, enabling attackers to execute arbitrary PHP code without authentication. This breach allows the creation of rogue administrator accounts by exploiting the Complex Calculation feature of the plugin. Hackers have been exploiting this vulnerability in the wild since April 13, despite a patch having been released on March 18.

**2) Why it matters:**

This vulnerability is significant because it threatens the security and integrity of websites using the Everest Forms Pro plugin. The ability for attackers to gain administrator access permits them to alter site content, install malicious plugins, and exfiltrate sensitive data. This can lead to financial losses, reputational damage, and the exploitation of visitor information, which is especially critical for transactional sites utilizing these forms for payments and personal data collection.

**3) What actions should be taken:**

Web administrators should immediately update the Everest Forms Pro plugin to the latest patched version. They are advised to monitor and review user accounts, specifically for any unknown administrator accounts named "diksimarina." Blocking the IP addresses 202.56.2[.]126 and 209.146.60.26 may help prevent further exploitation attempts. Administrators should also review server logs for unusual activities and consider deploying security plugins, such as Wordfence, to detect and block malicious traffic. Regular vulnerability assessments and patch management processes can also mitigate future risks.
