Original Article: https://www.bleepingcomputer.com/news/security/sap-fixes-three-critical-vulnerabilities-across-multiple-products/

**1) What happened:**

SAP released its December security updates to fix 14 vulnerabilities, including three critical ones, impacting various products. The most severe, CVE-2025-42880, is a code injection vulnerability in SAP Solution Manager ST 720, allowing attackers system control. Another critical flaw, CVE-2025-55754, affects SAP Commerce Cloud due to multiple Apache Tomcat vulnerabilities. Additionally, CVE-2025-42928, a deserialization vulnerability in SAP jConnect, could enable high-privileged remote code execution. These updates aim to address security risks in heavily integrated SAP systems across enterprises.

**2) Why it matters:**

The vulnerabilities pose significant risks, as SAP systems are integral to managing sensitive data and business operations within enterprises globally. The identified flaws, especially those enabling code execution and system control, threaten the confidentiality, integrity, and availability of critical business systems. Businesses relying on SAP for e-commerce, database connectivity, and system management could suffer substantial operational and reputational damage if exploited. Therefore, addressing these vulnerabilities is crucial to maintaining robust cybersecurity posture and protecting against potential attacks.

**3) What actions should be taken:**

Organizations using affected SAP products should prioritize applying the security patches provided in the December update to mitigate these risks. Reviewing and enhancing system monitoring and input validation processes might help identify any anomaly indicative of attempted exploitation. Additionally, conducting a comprehensive security assessment of their SAP environments can help identify and secure other potential vulnerabilities. By staying informed on SAP's security bulletains and adopting proactive security strategies, organizations can better safeguard their critical systems from attackers.
