Original Article: https://www.bleepingcomputer.com/news/microsoft/microsoft-extends-windows-server-2022-hotpatching-until-october-2027/

**1) What happened:**  
Microsoft has extended the hotpatching support for Windows Server 2022 Datacenter: Azure Edition until October 2027. Hotpatching enables Windows to deploy security updates without requiring system reboots, maintaining uptime and minimizing disruptions. This extension applies to systems enrolled in these updates, and the cadence of updates will remain unchanged. The extension provides an additional year of hotpatching support beyond the mainstream end date of October 2026, leading up to the extended support endpoint of October 2031 for Windows Server 2022.

**2) Why it matters:**  
This extension is significant as it allows organizations using Windows Server 2022 Datacenter: Azure Edition to continue benefiting from uninterrupted security updates for an additional year without the need for server restarts. With the increasing reliance on continuous system availability and the critical nature of timely security updates, hotpatching plays a crucial role in mitigating vulnerabilities and maintaining operational continuity. Businesses can leverage this advancement to ensure better compliance with security standards and minimize disruptions, especially for systems requiring high uptime.

**3) What actions should be taken as a result of this information:**  
Organizations using Windows Server 2022 should evaluate their current systems to determine eligibility for hotpatching and consider enrolling in the hotpatch update service if they are not already. Security teams should document this extension in their security policies and update lifecycle plans, ensuring they capitalize on the prolonged support. Additionally, IT departments should explore integrating hotpatching into broader security update strategies to maximize uptime and improve resilience against vulnerabilities. Training sessions could also be arranged for IT teams to stay informed about utilizing this feature effectively.
