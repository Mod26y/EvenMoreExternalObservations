1) What happened: Microsoft released a security update for Windows 11 in November 2025, enabling native support for third-party passkey management applications. This allows users to manage passkeys through apps like 1Password and Bitwarden using a new API that enhances passwordless authentication. Passkeys, adhering to FIDO2/WebAuthn standards, replace traditional passwords with a secure mechanism using cryptographic keys, enhancing security and user convenience. Authentication now involves a challenge to Windows, verified via Windows Hello, which employs PIN and biometric authentication methods. This feature adds flexibility by incorporating Microsoft Password Manager natively into Windows, offering enhanced security benefits.

2) Why it matters: Passkey support signifies a shift towards more secure and user-friendly authentication methods, reducing reliance on traditional passwords vulnerable to phishing and other attacks. By supporting third-party passkey managers, Microsoft expands user options for securing online accounts and enhances interoperability across different platforms. This update indicates a broader adoption of passwordless solutions, potentially reducing data breaches caused by weak or compromised passwords.
