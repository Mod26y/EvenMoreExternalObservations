1) The incident involved attackers compromising Trust Wallet's browser extension right before Christmas, leading to the theft of approximately $7 million. Attackers added a malicious JavaScript to the compromised version 2.68.0 of the Chrome extension, which exfiltrated sensitive wallet data and allowed them to steal from nearly 3,000 cryptocurrency wallets. Trust Wallet confirmed the hack and prompted an update to version 2.69 to halt further thefts, emphasizing an external breach possibly through a leaked Chrome Web Store API key.
