**What happened:**  
Microsoft has rolled out the Windows 10 KB5099539 extended security update, incorporating security and bug fixes from the July 2026 Patch Tuesday, which addressed an unprecedented 570 vulnerabilities. Additionally, Microsoft extended the free Windows 10 Extended Security Updates program, allowing eligible users to receive updates until October 2027. This update affects systems by updating them to builds 19045.7548 and 19044.7548 for Windows 10 and Windows 10 Enterprise LTSC 2021, respectively. It primarily addresses existing security vulnerabilities and bugs, such as issues with OLE Automation, File Explorer, and Secure Boot.

**Why it matters:**  
The update is significant as it tackles a record-breaking number of vulnerabilities, including two exploited zero-days, that could otherwise be leveraged by threat actors to compromise systems. By extending the free security updates, Microsoft is helping users protect their outdated systems against emerging threats without needing immediate system upgrades. This prolonged support is crucial for maintaining cybersecurity defenses and ensuring older systems are not left vulnerable, which is particularly important for organizations that have not yet transitioned to newer operating systems.

**What actions should be taken:**  
Organizations should prioritize installing these updates to safeguard their systems against the identified vulnerabilities. IT teams are encouraged to confirm compatibility with existing software, particularly third-party TDI transports, to prevent disruptions. Administrators should review and adjust their Group Policy settings for Remote Desktop Protocol security to align with new guidance. Monitoring the Windows System event logs for AFD Event ID: 16003 can help identify any impacted TDI transports. Transitioning from SHA-1 to stronger algorithms like SHA-256 for RDP security is advisable to future-proof systems against potential deprecations.
