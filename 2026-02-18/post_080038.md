Original Article: https://www.bleepingcomputer.com/news/security/notepad-plus-plus-boosts-update-security-with-double-lock-mechanism/

Notepad++ implemented a new "double-lock" mechanism to enhance the security of its update process after a supply-chain compromise. This development follows a campaign linked to a Chinese threat group, which compromised Notepad++'s update infrastructure by redirecting update requests to malicious servers. The double-lock mechanism involves verifying signed installers from GitHub and checking digitally signed XML from the notepad-plus-plus.org domain, mitigating risks from previous exploitative activities.

This mechanism is critical as it addresses security vulnerabilities in the software's update process, which can be a gateway for attackers to deploy malicious payloads to users. By enhancing the security of updates, Notepad++ reduces the risk of similar future supply-chain attacks that could compromise sensitive data and systems, highlighting the importance of robust software supply chain security.

Organizations should encourage all Notepad++ users to upgrade to version 8.9.2 to benefit from improved security measures. They should ensure that installers are downloaded exclusively from the official domain to avoid exposure to compromised software. Additionally, users and system administrators need to stay informed about updates, particularly those involving critical infrastructure changes that enhance security.
