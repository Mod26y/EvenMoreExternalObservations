Original Article: https://www.bleepingcomputer.com/news/microsoft/microsoft-now-force-upgrades-unmanaged-windows-11-24h2-pcs/

**1) What Happened:**
Microsoft began force-upgrading unmanaged Windows 11 24H2 Home and Pro editions to version 25H2 using machine learning-based rollout, bypassing the need for direct user or IT intervention. With Windows 11 24H2 nearing its end of support by October 2026, devices on this version will stop receiving necessary updates, security patches, and technical support. This upgrade is set to ensure continued security enhancements and functionality, with users given options to manually check for updates or choose when to restart after automatic upgrades.

**2) Why it Matters:**
This force-upgrade initiative by Microsoft is critical as it highlights the necessity for systems to stay updated within the latest support lifecycle to mitigate security vulnerabilities. With unmanaged devices no longer receiving necessary updates post-2026, there is a risk of increased cybersecurity threats without transitioning to supported versions. Keeping software up-to-date is a key element of an organization's cybersecurity posture, impacting device security, user data integrity, and overall system performance.

**3) Actions to be Taken:**
Organizations should inventory devices to identify any unmanaged systems running outdated Windows versions and develop a plan for timely upgrades. Users should be informed about the update process and any potential impacts on their daily operations. Engaging IT departments for more proactive management of updates can prevent automatic, unplanned upgrades that might disrupt workflows. Additionally, IT teams should review provided support documentation and guides to address any problems arising during the update process, ensuring a smooth transition to Windows 11 25H2.
