Original Article: https://www.bleepingcomputer.com/news/security/critical-marimo-pre-auth-rce-flaw-now-under-active-exploitation/

**1) What happened:** A critical remote code execution (RCE) vulnerability in the Marimo open-source reactive Python notebook platform has been exploited by hackers shortly after its disclosure. The flaw, tracked as CVE-2026-39987, allows unauthenticated users to execute code remotely, with attackers exploiting this to exfiltrate sensitive information. The issue stems from an insecure WebSocket endpoint which permits unauthorized interactive terminal access. Notably, the vulnerability was addressed with Marimo’s release of version 0.23.0, which implements fixes to prevent unauthorized access.

**2) Why it matters:** This exploitation is significant because Marimo is used by a wide array of data professionals, including data scientists and developers, potentially exposing sensitive data to unauthorized parties. Given its widespread use, the vulnerability has a substantial impact on the security of countless systems globally. The flaw’s immediate exploitation following its disclosure underscores the critical need for swift action on security advisories and the human resource and economic impacts of data breaches, including potential data loss, legal ramifications, and reputational damage.

**3) What actions should be taken:** Organizations using Marimo should prioritize upgrading to version 0.23.0 to mitigate this critical vulnerability. It is advisable for users to monitor WebSocket connections closely, particularly to ‘/terminal/ws,’ and employ firewall protections to restrict external access. In situations where upgrading is not feasible, users can block or disable access to the vulnerable endpoint to mitigate risk. Additionally, rotating exposed credentials and secrets is recommended to further safeguard against unauthorized access and data breaches. Implementing comprehensive security assessments and awareness campaigns about the use of security patches can bolster organizational resilience against similar threats.
