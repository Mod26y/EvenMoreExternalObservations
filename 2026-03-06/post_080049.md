Original Article: https://www.bleepingcomputer.com/news/security/wordpress-membership-plugin-bug-exploited-to-create-admin-accounts/

The article highlights a severe vulnerability in the WordPress User Registration & Membership plugin. Hackers are exploiting this flaw, CVE-2026-1492, which permits them to create unauthorized administrator accounts. Given the critical severity rating of 9.8, this vulnerability poses substantial risks, as admin accounts could allow attackers to change site settings, steal data, embed malware, and potentially disrupt site operations. The vulnerability affects versions up to 5.1.2, and a patch is available in subsequent versions.

This vulnerability matters significantly because it impacts more than 60,000 WordPress sites, primarily used for membership management. As administrator accounts grant comprehensive access, exploiting this flaw can severely compromise site integrity, data security, and user trust. This kind of access may lead to data breaches and the distribution of malicious content to users, further leveraging the compromised site for broader malicious activities.

In response, WordPress site administrators should prioritize updating the User Registration & Membership plugin to version 5.1.4, where the fix is implemented. If updating immediately is challenging, temporarily disabling or uninstalling the plugin can help mitigate risks. Additionally, site owners should reinforce security measures, such as employing web application firewalls like Wordfence, to monitor and block exploitation attempts actively. Regularly reviewing plugin configurations and permissions can also prevent similar issues.
