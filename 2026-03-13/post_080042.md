Original Article: https://www.bleepingcomputer.com/news/security/veeam-warns-of-critical-flaws-exposing-backup-servers-to-rce-attacks/

**What happened:**
Veeam Software has released patches for several critical vulnerabilities in its Backup & Replication (VBR) solution. These vulnerabilities, including four critical remote code execution (RCE) flaws, allow attackers with low privileges to execute remote code on backup servers. One critical flaw specifically allows execution as the postgres user. High-severity vulnerabilities were also patched, which could enable privilege escalation, credential extraction, and file manipulation on Windows-based servers. These patches stem from discoveries in internal testing or reports through HackerOne and are included in VBR versions 12.3.2.4465 and 13.0.1.2067.

**Why it matters:**
Veeam's Backup & Replication software is a crucial tool for data backup and recovery used widely, especially among large enterprises and managed service providers. Ransomware and threat groups frequently target these servers owing to their strategic value in both lateral movement and disruption of backup processes. Notable cybercriminal groups like FIN7 have exploited VBR vulnerabilities in the past. The timely patching of these vulnerabilities is essential to prevent exploitation, especially as attackers commonly develop exploits soon after patches are released, jeopardizing unpatched systems.

**What actions should be taken:**
Organizations using Veeam's VBR should promptly upgrade to the latest versions, 12.3.2.4465 or 13.0.1.2067, to mitigate the risk posed by these vulnerabilities. Continuous monitoring for any signs of ongoing exploitation is advisable. Additionally, IT departments should review backup server access privileges, ensuring minimal access rights are given to users. Assessing overall security posture through penetration testing or vulnerability assessments can also help identify and rectify potential weaknesses. Emphasizing the need for regular updates and patch management can further safeguard systems against future threats.
