Original Article: https://www.bleepingcomputer.com/news/security/malicious-jetbrains-marketplace-plugins-steal-ai-api-keys-from-developers/

1) What happened: At least 15 malicious plugins on the JetBrains Marketplace were identified as stealing AI API keys from developers. These plugins, masquerading as AI coding assistants and other tools, were discovered by Aikido Security. They exfiltrate API keys by transmitting them to an external server upon user application. With approximately 70,000 installations, the plugins have been active since October 2025, with new ones appearing as recently as June 2026. The plugins operate normally but secretly capture and transmit the developers' API keys.

2) Why it matters: This incident highlights a serious vulnerability in trust and security within reputable developer platforms like the JetBrains Marketplace, which developers rely on for trustworthy tools. The theft of API keys poses significant risks, including unauthorized access, financial charges, and intellectual property loss, especially given the wide adoption of AI tools in software development. The presence of such malicious plugins could lead to broader implications, such as eroding trust in marketplace security and potentially facilitating further cyber threats and exploitation.

3) What actions should be taken: Developers should review and possibly revoke compromised API keys immediately to prevent misuse. Moreover, enhancing security measures when downloading plugins is essential, including using monitoring tools and scrutinizing plugin background activities. JetBrains should expedite their review and removal processes for malicious plugins and improve Marketplace vetting procedures. Continuous user awareness and training on recognizing malicious activities and secure coding practices are also crucial. Engaging with security research groups like Aikido can bolster ongoing monitoring and threat intelligence collaboration.
