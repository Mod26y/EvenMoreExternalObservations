This flaw is critical because it affects around 10,000 WordPress sites using the plugin, compromising website security by granting admin access to unauthorized users. This could lead to data breaches, defacement, or misuse of website functionalities. With over 48,400 blocked attempts, the rapid exploitation following the public disclosure highlights the urgency for affected website administrators to assess and strengthen their security frameworks.

Website owners using King Addons should update to the patched version 51.1.35, which fixes the flaw. Additionally, they should monitor WordPress logs for unauthorized admin accounts, and suspicious IP addresses and consider preventive measures like implementing firewalls and regular security audits. A similar approach should be applied to the Advanced Custom Fields: Extended plugin by upgrading to version 0.9.2 to mitigate remote code execution risks from CVE-2025-13486.
