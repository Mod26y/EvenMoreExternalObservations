Original Article: https://www.bleepingcomputer.com/news/security/malicious-edge-extension-abuses-native-messaging-as-bridge-to-malware/

1) The incident involved a malicious Microsoft Edge extension, named "Edgecution," that exploited Chrome's Native Messaging protocol to bypass browser sandbox restrictions and deploy a backdoor on the system. Attackers initially posed as IT support on Microsoft Teams, directing users to a phony update page. The malware was linked to the Payouts Kings ransomware and executed through scripts from a fake Microsoft site. This extension facilitates communication between a browser and native applications, enabling the execution of destructive commands on infected systems.

2) This event is significant because it underscores evolving threats exploiting browser features to circumvent security measures. By leveraging popular applications like Microsoft Teams and Microsoft Edge, the attackers can blend in with legitimate traffic, making detection difficult. The sophistication of the attack, using a combination of social engineering and technical maneuvers, highlights the growing threat of ransomware operations that target both user trust and technological vulnerabilities. It emphasizes the need for heightened vigilance in monitoring and managing browser extensions and their interaction with native applications.

3) Organizations should enhance their security protocols by implementing robust monitoring of browser extensions and establishing strict governance over native messaging host configurations. Employee training initiatives should be intensified to help staff recognize and report phishing attempts, especially those using IT support as a guise. Additionally, security teams can leverage threat intelligence reports like Zscaler's IoC list to update and refine their security systems, ensuring swift detection and response to similar threats. Regular security assessments, including breach and attack simulations, could further ensure that security defenses remain resilient against such complex attack strategies.
