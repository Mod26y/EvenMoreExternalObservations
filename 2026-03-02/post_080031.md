Original Article: https://www.bleepingcomputer.com/news/security/quicklens-chrome-extension-steals-crypto-shows-clickfix-attack/

1) **What happened**: The QuickLens Chrome extension, initially recognized for enabling Google Lens searches, was compromised after its ownership was transferred. The new owner released a malicious update on February 17, 2026, containing scripts that performed ClickFix attacks and stole information from approximately 7,000 users. The extension requested new permissions to strip browser security headers, allowing the injection and execution of harmful scripts on all visited pages. These scripts triggered fake Google Update prompts that led to malware downloads. They also targeted crypto wallets and sensitive information, utilizing injected JavaScript to steal data.

2) **Why it matters**: This incident highlights the danger of browser extensions, which can be easily updated with malicious code after ownership changes. Users may unknowingly expose their systems to malware and data theft due to compromised security settings. The attack underscores the need for vigilant security practices, particularly as extensions become more sophisticated in evading detection. Such incidents can cause significant financial losses, particularly with the theft of cryptocurrency assets, and raise concerns about data privacy and the security of online activities.

3) **What actions should be taken**: To mitigate the risk of similar incidents, organizations and individuals should regularly audit browser extensions for permissions changes and suspicious activity. Implementing an extension management policy that restricts installation permissions can strengthen security. Users should immediately remove the QuickLens extension, conduct comprehensive malware scans, and change passwords for sensitive accounts. For those using affected cryptocurrency wallets, transferring funds to new wallets could prevent potential theft. Additionally, maintaining updated cybersecurity awareness and deploying effective endpoint security tools can better protect against such evolving threats.
