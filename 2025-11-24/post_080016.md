**1) What happened:**  
Google has enhanced interoperability between its Quick Share feature on Android and Apple's AirDrop, allowing for file sharing between Pixel devices and iPhones. Initially, this functionality is limited to Pixel 10-series, but plans are in place to extend support to additional Android models. Both systems have traditionally been closed ecosystems, restricting cross-platform file sharing. The innovation incorporates security protocols, verified by cybersecurity firm NetSPI, and uses Rust for enhanced memory safety. Users can share data device-to-device without intermediaries and must manually confirm the receiving device to ensure secure data transfer.
