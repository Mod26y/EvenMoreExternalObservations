Original Article: https://www.bleepingcomputer.com/news/security/bitwarden-adds-support-for-passkey-login-on-windows-11/

1) **What happened:** Bitwarden has launched a new feature that enables users to log into Windows 11 using passkeys stored in the manager's encrypted vault instead of passwords, offering phishing-resistant authentication. This functionality is available across all Bitwarden subscription plans. To utilize this feature, devices must be Entra ID-joined, have FIDO2 security key sign-in enabled, and hold a registered Entra ID passkey. The passkey login method employs cryptographic credentials stored in Bitwarden, minimizing credential exposure.

2) **Why it matters:** This development is significant because it enhances security by eliminating the use of passwords, which are susceptible to phishing attacks. Passkey authentication in Windows 11 increases security while simplifying the authentication process across devices. As it leverages open-source technology, it provides an accessible, scalable solution to organizations, including governmental entities, looking to minimize cybersecurity risk and improve their defense posture against sophisticated threats keen on exploiting user credentials.

3) **What actions should be taken:** Organizations should evaluate the feasibility of integrating passkey login features within their IT infrastructure, ensuring devices are compatible and meet the necessary conditions. It would be beneficial to train IT staff and end-users on this new method of authentication, emphasizing its security advantages. Additionally, they should ensure that policies and procedures surrounding authentication and access management are updated to incorporate these changes, and monitor the transition to passkey use to resolve any issues promptly. Regular reviews of endpoint security configurations will also be critical to ensure ongoing compliance.
