Original Article: https://www.bleepingcomputer.com/news/security/new-fortinet-forticlient-ems-flaw-cve-2026-35616-exploited-in-attacks/

1) The FortiClient Enterprise Management Server (EMS) is facing a critical vulnerability identified as CVE-2026-35616. It allows unauthorized attackers to execute code remotely through specially crafted requests. Fortinet released an emergency patch for the flaw that affects FortiClient EMS versions 7.4.5 and 7.4.6 and has been exploited actively. The vulnerability bypasses pre-authentication API access controls, enabling attackers to skip authentication and authorization entirely.

2) This vulnerability is significant as it poses a direct threat of remote code execution, potentially leading to unauthorized access to sensitive data and systems. Considering Fortinet’s widespread use, particularly in the US and Germany where over 2,000 exposed instances have been reported, the vulnerability's exploitation could impact numerous organizations, leading to a compromise in data integrity and operational disruptions.

3) Organizations using affected versions of FortiClient EMS need to promptly implement the recommended hotfixes or plan to upgrade to version 7.4.7 once it becomes available. It's also advisable to review firewall and access control policies, continuously scan networks for potential intrusions, and apply a layered security approach with regular updates. Staying vigilant and strengthening incident response strategies will help mitigate future exploitation risks.
