Original Article: https://www.bleepingcomputer.com/news/security/new-macos-stealer-campaign-uses-script-editor-in-clickfix-attack/

In a recent cybersecurity incident, attackers are using a new campaign to target macOS users with Atomic Stealer malware by exploiting the macOS Script Editor in an advanced "ClickFix" attack. This scam involves fake Apple-themed sites that have system cleanup instructions, deceptively prompting users to open the macOS Script Editor and execute malicious scripts. These scripts run concealed commands that eventually deploy the Atomic Stealer, a malware designed to extract sensitive information like passwords, financial data, and cryptocurrency wallet details from the infected system.

This cyber incident is significant because it highlights vulnerabilities in the macOS ecosystem, specifically through trusted built-in applications like Script Editor. The attack demonstrates the effectiveness of social engineering by mimicking legitimate Apple services, tricking users into executing harmful commands and downloading malware. As the Atomic Stealer poses a risk to both personal and corporate sensitive information, understanding and addressing this threat is crucial for users of macOS.

Organizations and macOS users should increase their awareness of such deceptive tactics and treat any unexpected Script Editor prompts as potentially harmful. To mitigate these threats, relying on official Apple troubleshooting guides and community support forums is advised. Additionally, security teams should consider conducting regular security awareness training for users to recognize and handle phishing attempts. Implementing advanced endpoint protection solutions can further help detect and block malicious activities associated with these types of attacks.
