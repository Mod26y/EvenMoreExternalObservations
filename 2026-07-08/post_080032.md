Original Article: https://www.bleepingcomputer.com/news/security/hidden-backdoor-in-tenda-router-firmware-grants-admin-access/

1) What happened: A hidden backdoor was discovered in multiple Tenda router firmware versions, identified as CVE-2026-11405. This flaw allows attackers to gain administrative access to the device’s web management panel by exploiting an undocumented authentication mechanism. By using an alternate password from the 'sys.rzadmin.password' configuration, any user can gain admin access, regardless of the entered username. The issue remains unaddressed as Tenda has been unreachable, leaving the routers vulnerable to unauthorized control.

2) Why it matters: The vulnerability is significant because it can grant attackers full administrative access to the affected Tenda routers. With such control, they can reconfigure the device, potentially altering network settings and disabling security features, thereby compromising the entire local network. This makes the routers attractive targets for attackers, especially botnets exploiting router flaws, posing significant risks to confidentiality, integrity, and availability of network resources.

3) Actions to be taken: Users should disable the remote web management panel to prevent remote access to the vulnerable interface. Restricting local network exposure by changing the default LAN IP address can also help reduce the risk of unauthorized access. Continuous monitoring for unusual activity and applying network segmentation can further mitigate risks. Organizations using these devices should consider replacing them if a patch does not become available, to safeguard critical network infrastructure.
