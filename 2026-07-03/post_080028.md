Original Article: https://www.bleepingcomputer.com/news/security/consentfix-and-clickfix-how-microsoft-365-accounts-are-hijacked-in-3-seconds/

**What Happened:** The article highlights two sophisticated attack vectors, ClickFix and ConsentFix, that exploit user habits to hijack Microsoft 365 accounts. ClickFix tricks users into executing malicious commands via fake verification prompts, while ConsentFix manipulates OAuth consent flows, tricking users into surrendering session tokens. These attacks do not exploit vulnerabilities but instead capitalize on conditioned user behaviors and authentic-looking prompts. The ease of executing these attacks has been propagated through a Russian cybercrime forum, lowering the barrier for cybercriminals to conduct similar breaches.

**Why It Matters:** This form of attack is critical because it targets the weakest link in security: human behavior. By mimicking legitimate processes, these attacks bypass traditional security defenses like antivirus programs and firewalls, successfully accessing user accounts without exploiting software vulnerabilities. Given the structured dissemination of these techniques on cybercrime platforms, there’s a heightened risk of increased attacks. For organizations, particularly those using Microsoft 365, this represents a significant threat to sensitive data, operational integrity, and potential compliance violations resulting from unauthorized data access.

**What Actions Should Be Taken:** Organizations should emphasize user training to question and verify unusual requests, especially those involving credential processes or link drag-and-drops. Additionally, enhancing endpoint detection and response (EDR) systems to flag anomalous behaviors such as unusual PowerShell activity or unexpected login locations can help detect these attacks early. Implementing real-time identity monitoring may also help intercept unauthorized session access. Engagement in continuous learning forums like Tradecraft Tuesday could further bolster an organization's knowledge of evolving cyberthreats and improve defense strategies.
