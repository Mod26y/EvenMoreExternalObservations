Original Article: https://www.bleepingcomputer.com/news/security/smart-slider-updates-hijacked-to-push-malicious-wordpress-joomla-versions/

1) The Smart Slider 3 Pro plugin for WordPress and Joomla was compromised when hackers hijacked its update system to deploy a malicious version (3.5.1.35) containing multiple backdoors. These backdoors allowed for hidden admin account creation, sensitive data theft, command execution without authentication, and persistent malware infiltration across multiple components. The developer has urged users to update to version 3.5.1.36 or revert to versions before 3.5.1.35 to rectify the situation. The attack is widespread, potentially impacting over 900,000 websites using the plugin for responsive slider creation.

2) This incident underscores the critical vulnerabilities inherent in third-party plugins that can serve as vectors for widespread attacks on content management systems such as WordPress and Joomla. It highlights the importance of maintaining rigorous update controls and monitoring systems in safeguarding website infrastructure against unauthorized access and data theft. Given the potential for secondary attacks through compromised admin accounts and hidden backdoors, any affected systems could pose significant risk to associated services and user data.

3) Administrators should prioritize updating the Smart Slider plugin to the safe version. If the site is compromised, they ought to delete malicious users, files, and database entries, reestablish the WordPress core, and reinstall plugins and themes from reliable sources. Rotating all credentials and regenerating WordPress security keys are advised practices. Additionally, implementing two-factor authentication, maintaining updated components, restricting admin access, and continually monitoring for malware are recommended to enhance security posture and prevent future breaches. Regular backups and employing comprehensive patch management can also mitigate risks.
